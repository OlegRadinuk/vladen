# Контекст проекта: Владен (строительная компания, Крым)

Ты помогаешь разрабатывать сайт на Next.js 16 + Tailwind + TypeScript.

## Общие принципы

- **Mobile first** — сначала верстка под 375px, потом планшет (768px), потом десктоп (1280px+)
- **Адаптивность** — все отступы через `px-4 sm:px-6 lg:px-8`, контейнеры через `max-w-7xl mx-auto`
- **Безопасность** — все токены и ключи в `.env.local`, не коммитить `.env` нужно глобавльная безопассность на всем пути проекта
- **SEO** — метатеги на каждой странице, sitemap.xml, robots.txt

## Стили и эффекты

- **Hover** — без прыжков, только glow, смена цвета границы, появление оранжевого акцента
- **Анимации появления** — при скролле (выезжание слева/справа), но без фанатизма
- **Параллакс** — только в Hero (движение фона за мышью)
- **Карточки услуг** — фото вместо иконок, при наведении оранжевая полоска снизу

## Форма и Telegram

- Форма: имя + телефон
- Отправка на `/api/telegram`
- Валидация телефона (минимальная)
- Защита: проверка origin, только POST

## SEO

- На каждой странице: `title`, `description`, `openGraph`
- `sitemap.xml` и `robots.txt` в public

## Что нужно сделать в первую очередь (MVP)

1. Корневой layout (Header + Footer)
2. Главная страница (Hero, Услуги, Кейс, Почему мы, Калькулятор, Контакты)
3. Страницы: Услуги, Проекты, О компании, Контакты
4. Форма + Telegram
5. SEO база

## Запуск

После создания проекта: `npm run dev`
Сайт доступен на `http://localhost:3000`
